EA0F: Enemy Colour Cycle Table
Eight attribute bytes used by Process_Wall_Bouncing_Enemy and Animate_Wall_Bouncing_Enemy_Death to set the entity display colour on each animation frame. Indexed by the low 3 bits of the colour cycle counter at C4E5.
