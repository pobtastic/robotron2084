E520: Process Cluster Entity
Per-frame entity handler for the cluster entities spawned when a tracked wave enemy promotes. Advances three bytes to a countdown timer, decrements it, and redraws the entity via Clear_Colour_Bar_Column_26 while the timer is still running. When the timer reaches zero, calls Erase_Entity_Sprite to erase the entity sprite. If the animation pass counter (two bytes back) is also exhausted, detaches the entity and scores; otherwise increments the pass counter, redraws via Draw_Entity_Sprite, sets the animation delay to 04 and jumps to Tick_Entity_Colour_Timer_Advance. On further passes the handler transitions to Process_Cluster_Entity_Homing, which searches for player contact via Find_Colliding_Entity and triggers a kill via Process_Player_Tracking_Enemy if found, or else applies directional movement toward the player and draws via Draw_Narrow_Entity_Sprite.
