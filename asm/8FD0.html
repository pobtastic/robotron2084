8FD0: Control Code Dispatch Table
Two-byte handler address table for control codes 10–1A indexed by the control code zero-indexed from 10. Each entry is the address of the parameter handler installed into PrintState_Handler by PrintHandler.
ControlCode_DispatchTable 8FD0 DEFW PrintHandler_Ink 10: INK colour — parameter handler at PrintHandler_Ink.
8FD2 DEFW PrintHandler_Paper 11: PAPER colour — parameter handler at PrintHandler_Paper.
8FD4 DEFW PrintHandler_Flash 12: FLASH — parameter handler at PrintHandler_Flash.
8FD6 DEFW PrintHandler_Bright 13: BRIGHT — parameter handler at PrintHandler_Bright.
8FD8 DEFW PrintHandler_Unsupported 14: Unsupported — parameter consumed and discarded via PrintHandler_Unsupported.
8FDA DEFW PrintHandler_Unsupported 15: Unsupported — parameter consumed and discarded via PrintHandler_Unsupported.
8FDC DEFW PrintHandler_PrintAt_X 16: PRINT AT column — parameter handler at PrintHandler_PrintAt_X.
8FDE DEFW PrintHandler_Tab 17: TAB — parameter handler at PrintHandler_Tab.
8FE0 DEFW PrintHandler_LeftMargin 18: Set left margin — parameter handler at PrintHandler_LeftMargin.
8FE2 DEFW PrintHandler_RightMargin 19: Set right margin — parameter handler at PrintHandler_RightMargin.
8FE4 DEFW PrintHandler_SelectFont 1A: Select font — parameter handler at PrintHandler_SelectFont.
