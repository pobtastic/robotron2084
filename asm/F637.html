F637: Draw Attract Title
Used by the routine at Attract_Mode_Loop.
Attract-mode title setup after the screen is cleared: writes two parameter sets to TitleBlit_Params–C4FC and OR-blits ROM logos from 6D7D and 7005 via BlitTitleStrips, fills C0 attribute cells from 5920 with YELLOW (yellow ink on black paper), then prints Messaging_Credits (copyright / Williams credits) via PrintMessaging. Called from Attract_Mode_Loop after DrawIntroSprites.
DrawAttractTitle F637 LD HL,$C4F9 HL=TitleBlit_Params (parameter block).
First bitmap: parameters 28 20 18 43 then source 6D7D.
F63A LD (HL),$28 Write 28 to *TitleBlit_Params.
F63C INC HL Write 20 to *C4FA.
F63D LD (HL),$20
F63F INC HL Write 18 to *C4FB.
F640 LD (HL),$18
F642 INC HL Write 43 to *C4FC.
F643 LD (HL),$43
F645 LD HL,$6D7D HL=6D7D (first title / logo bitmap in ROM).
F648 CALL BlitTitleStrips Call BlitTitleStrips.
F64B LD HL,$C4F9 Write 4A to *TitleBlit_Params.
Second bitmap: parameters 4A 58 0A 69 then source 7005.
F64E LD (HL),$4A
F650 INC HL Write 58 to *C4FA.
F651 LD (HL),$58
F653 INC HL Write 0A to *C4FB.
F654 LD (HL),$0A
F656 INC HL Write 69 to *C4FC.
F657 LD (HL),$69
F659 LD HL,$7005 HL=7005 (second title / logo bitmap in ROM).
F65C CALL BlitTitleStrips Call BlitTitleStrips.
Attribute band for the credits line (C0 cells, attribute YELLOW).
F65F LD HL,$5920 HL=5920 (start within the attribute file 5800–5AFF).
F662 LD B,$C0 Set a loop counter in B for C0 attributes.
DrawAttractTitle_FillAttrs F664 LD (HL),$06 Write YELLOW (06) to *HL.
F666 INC HL Advance HL.
F667 DJNZ DrawAttractTitle_FillAttrs Decrease counter and loop back to DrawAttractTitle_FillAttrs until the band is filled.
F669 LD HL,$F670 Call PrintMessaging using Messaging_Credits.
F66C CALL PrintMessaging
F66F RET Return.
