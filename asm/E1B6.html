E1B6: Handle Horizontal Enemy Entity
Per-frame entity handler for the horizontal-moving wave enemy, written into entity records by Spawn_Horizontal_Wave_Enemies. Checks the wave start delay at C533; if still counting, skips forward via Process_Entity_Record_Counter_Up_Next6. Otherwise erases the sprite, tests player contact via HitTestBounds8AtHL (triggering death via Strip_Attributes_And_Fade if touching), searches the wave enemy list for a collision partner via Find_Colliding_Wave_Enemy, and repositions the entity by applying a directional delta from the table at Direction_Delta_Table. Spawns the entity at its new position via Draw_Wide_Entity_Sprite and advances the entity record pointer via Process_Entity_Record_Next6.
