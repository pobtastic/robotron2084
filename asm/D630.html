D630: Erase Entity Sprite
Used by the routines at Main_Game_Loop, Clear_Colour_Bar_Column, Process_Entity_Record_Fast, Retire_Expired_Entity and Process_Cluster_Entity.
Entry point for erasing an entity sprite. Sets erase mode then falls through to Draw_Entity_Sprite_Render. In erase mode the attribute write is skipped.
Erase_Entity_Sprite D630 EXX Set erase mode (attribute write disabled).
D631 LD D,$00
D633 EXX
D634 JP Draw_Entity_Sprite_Render Jump to Draw_Entity_Sprite_Render.
