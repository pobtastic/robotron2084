E492: Tick Entity Colour Timer
Entity handler that drives a per-entity colour timer. Each frame it decrements the timer byte; if still non-zero, extracts the animation frame index from the low 3 bits and calls Clear_Colour_Bar_Column_26 to refresh the entity's attribute colours, then advances the entity pointer by 0B bytes via Tick_Entity_Colour_Timer_Advance. When the timer reaches zero, falls through to Retire_Expired_Entity to erase and retire the entity. Written into entity records by Kill_Tracked_Enemy_Entity.
Tick_Entity_Colour_Timer E492 DEC (HL) Decrement the entity timer; if still live, extract the low 3 bits as the animation frame index and refresh the entity's attribute colours via Clear_Colour_Bar_Column_26.
