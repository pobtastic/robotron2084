8DCC: Print Dispatch
Used by the routines at PrintMessaging, PrintPlayerScore, PrintWaveNumber, PlayerDeath, Print_BCD_Score, Draw_Score_Entry and Display_High_Score_Screen.
Calls the active print handler stored at PrintState_Handler to process one character or control code, preserving caller context across the call.
PrintDispatch 8DCC PUSH BC Stash BC on the stack.
8DCD EXX Switch to the shadow registers.
8DCE PUSH HL Stash HL and IX on the stack.
8DCF PUSH IX
8DD1 LD IX,$8000 Set IX to the print state base at PrintState_Y.
8DD5 LD HL,$8DDD Push Print_Dispatch_Return as the handler return address.
8DD8 PUSH HL
8DD9 LD HL,($800A) Read the current print handler address from PrintState_Handler.
8DDC JP (HL) Jump to the print handler.
