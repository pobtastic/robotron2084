F6EE: Read Keyboard Split
Control mode 05: reads both keyboard rows separately, returning the F7FE row in B and the EFFE row in C.
Read_Control_Keyboard_Split F6EE CALL Read_Keyboard_Rows Call Read_Keyboard_Rows to read both keyboard rows.
F6F1 AND %00011111 Mask the F7FE row result to five bits.
F6F3 LD B,A Copy the F7FE row result to B and load the EFFE row.
F6F4 LD A,C
F6F5 AND %00011111 Mask the EFFE row result to five bits.
F6F7 LD C,A Store the masked EFFE row result in C.
F6F8 POP HL Discard the stacked return address.
F6F9 RET Return.
