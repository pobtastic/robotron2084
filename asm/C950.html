C950: Draw Wide Entity Sprite
Used by the routines at Clear_Colour_Bar_Column, Process_Entity_Record_Fast, Spawn_Wave_Enemies, Handle_Horizontal_Enemy_Entity, Process_Tracked_Enemy_Entity, Spawn_Tracked_Enemy_Entities, Process_Wall_Bouncing_Enemy, Spawn_Wall_Bouncing_Enemies, Process_Replacement_Enemy_Entity and Process_Final_Phase_Enemy_Entity.
Draws or erases a wide entity's sprite on the display. Draw_Wide_Entity_Sprite sets draw mode so the entity colour is written to the attribute area. Erase_Wide_Entity_Sprite sets erase mode and skips the attribute write. Both paths enter Draw_Wide_Entity_Sprite_Render, which reads the entity type to look up sprite data from Sprite_Pixel_Data_Table, computes the screen position from the Y and X bytes, then XOR-blends sixteen pixel rows of sprite data onto the display across three bytes (24 pixels) wide.
Input
HL Entity record pointer
