F092: Draw Concentric Rings
Used by the routine at Draw_Colour_Border.
Draws D nested rectangular colour rings, each one cell inward from the last, by calling Draw_Colour_Ring with incrementing corner offsets.
Input
BC Starting corner offset (B = column, C = row)
D Number of rings to draw
Draw_Concentric_Rings F092 PUSH DE Stash the ring counter, draw the current colour ring via Draw_Colour_Ring, and restore.
F093 CALL Draw_Colour_Ring
F096 POP DE
F097 INC C Move the corner one cell inward on each side.
F098 INC B
F099 DEC D Repeat for each remaining ring.
F09A JR NZ,Draw_Concentric_Rings
F09C RET Return.
