F54D: Colour Cycle Table
Sequence of ZX Spectrum attribute byte values used to drive the colour cycling animation. Each byte specifies the colour to write; FF is the end-of-sequence sentinel that resets the pointer to the table start.
Colour_Cycle_Table F54D DEFB $01,$05,$02,$04,$03,$06,$FF
