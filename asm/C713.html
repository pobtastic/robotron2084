C713: Update Attribute Bar
Used by the routines at Main_Game_Loop, Clear_Colour_Bar_Column, RenderHUD and PlayerDeath.
Paints the HUD attribute border with *Border_Colour and writes the cycling colour from HelpBarAttributeSequence to 07 attribute cells at *Colour_Bar_Column. Checks the H key (help) and F key (restart). UpdateAttributeBar_FillBorder is also called directly by C5BD.
UpdateAttributeBar C713 LD HL,($C4E5) HL=*Colour_Cycle_Table_Pointer (colour cycle table pointer).
UpdateAttributeBar_FetchColour C716 LD A,(HL) Read the next colour byte; jump to UpdateAttributeBar_ResetTable if zero (table end).
C717 AND A
C718 JP Z,UpdateAttributeBar_ResetTable
C71B INC HL Advance the table pointer and save it back to *Colour_Cycle_Table_Pointer.
C71C LD ($C4E5),HL
