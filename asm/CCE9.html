CCE9: Process Enemy Entity
Used by the routine at Primary_Enemy_Entity_Handler.
Draws a wave enemy entity's sprite, updates its handler to the entity update completion routine, then checks for contact with the player. If the enemy touches the player, the entity is erased, removed from the list, the wave enemy count is decremented, and execution returns to the dispatcher.
Process_Enemy_Entity CCE9 PUSH HL Stash the entity record pointer, advance four bytes to the screen address field, and draw the entity sprite via Draw_Spawned_Entity.
