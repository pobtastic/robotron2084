C5C1: Check Score Award Life
Used by the routine at Main_Game_Loop.
Compares four bytes at Player_1_State with the template at ExtraLifeCompare_b0. Returns if any player byte is below the matching template byte. Otherwise increments *Lives, calls DrawLivesDisplay, adds BCD 02 to *ExtraLifeCompare_b1 with carry into *Lives. If *Active_Enemy_Count is non-zero, tests *FFFF and keyboard port FE then RST 08 (error code FF); if *Active_Enemy_Count is 00 jumps to CheckScoreAwardLife_Continue. At CheckScoreAwardLife_Continue: pop DE, Extra_Life_Flash, Stage_Next_Wave. Used from Erase_Narrow_Entity_Sprite.
