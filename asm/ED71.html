ED71: Check Any Key During Display
Used by the routines at Display_High_Score_Entry_Screen, Save_Or_Load_High_Scores and Attract_Mode_Loop.
Reads all keyboard rows simultaneously (port FE with selector 00) during the high score display loop. Forces the upper three bits high; if any key is pressed, increments A past FF discards the caller's return address, and restarts the high score display at Display_High_Score_Entry_Screen. Returns normally if no keys are held.
Check_Any_Key_During_Display ED71 XOR A Read all keyboard rows simultaneously.
ED72 IN A,($FE)
ED74 OR %11100000 Force bits 5-7 high.
ED76 INC A Return if no keys are pressed (A wraps to 00).
ED77 RET Z
ED78 POP HL Discard the caller's return address from the stack.
ED79 JP Display_High_Score_Entry_Screen Restart the high score display at Display_High_Score_Entry_Screen.
