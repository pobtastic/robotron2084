D486: Draw Life Icon Cells
Used by the routine at DrawLivesDisplay.
Draws the life icon cell pixels into the display file. Given an icon index in A, looks up the icon cell data address from the descriptor table, then for each cell computes the screen pixel address and OR-blits the icon pixels into the display file. Used by DrawLivesDisplay to repaint the lives display.
Input
A Icon index; selects the cell data from the icon descriptor table

D493 LD D,B
