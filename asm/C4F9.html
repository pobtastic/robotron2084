C4F9: Title Strip Blit Parameters
Four bytes read by BlitTitleStrips: *TitleBlit_Params and *C4FC bound the strip loop, *C4FA selects the screen row / band, *C4FB is the run length (LDIR) per strip. Filled by DrawAttractTitle before each BlitTitleStrips call.
TitleBlit_Params C4F9 DEFB $00 Strip loop initial index.
C4FA DEFB $00 Row / band parameter for the screen row address lookup table.
C4FB DEFB $00 Bytes per strip (LDIR count).
C4FC DEFB $00 Strip loop terminal index (stop when B reaches this value).
