ED93: Scan Score Display Attributes
Used by the routine at Display_High_Score_Entry_Screen.
Scans a contiguous block of attribute cells from DE to IX, applying the same colour substitution as Scan_And_Cycle_Screen_Attributes. Called during the name entry phase to keep the score display colours consistent.
Input
DE Start attribute address
IX End attribute address (exclusive)
Scan_Score_Display_Attributes ED93 LD A,(DE) Load the current attribute byte from DE.
ED94 LD HL,$EE5B Set up to search for this attribute in the substitution table at EE5B.
ED97 LD BC,$000A
ED9A CPIR Search for the attribute in the substitution table.
ED9C JR NZ,Scan_Score_Attr_Next Skip replacement if the attribute was not found.
ED9E LD A,(HL) Replace the attribute with its substitution byte.
ED9F LD (DE),A
Scan_Score_Attr_Next EDA0 INC DE Advance to the next attribute cell.
EDA1 PUSH IX Copy the end address from IX to HL.
EDA3 POP HL
EDA4 AND A Clear carry and test whether the range is exhausted.
EDA5 SBC HL,DE
EDA7 JR NZ,Scan_Score_Display_Attributes Loop back to Scan_Score_Display_Attributes until all cells in the range are processed.
EDA9 RET Return.
