F2F9: Draw Score Entry
Used by the routine at Display_High_Score_Screen.
Draws one score entry on the high score screen. Given the screen position in DE and the name data in HL, prints the rank number from C503, a closing parenthesis, the three-character name, then the full score digits via Print_BCD_Score. The alternate entry at Draw_Score_Entry_Compact uses tighter row spacing and a smaller column advance for the All Time Heroes table.
Input
DE Screen position (row in D, column in E)
HL Points to the three-character name for this score entry
