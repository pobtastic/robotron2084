D9E0: Dispatch Entity State Handler
Used by the routine at Main_Game_Loop.
Called once per frame by the main game loop at Main_Game_Loop. Reads the state flag at Entity_State_Flag; if 02 (complete) returns immediately. Otherwise loads the dispatch pointer from Entity_State_Pointer and jumps through it to the active entity state handler.
Dispatch_Entity_State_Handler D9E0 LD A,($C52F) Return if the entity state is complete (02).
D9E3 CP $02
D9E5 RET Z
D9E6 LD HL,($C521) Load the entity state dispatch pointer from Entity_State_Pointer and jump through it.
D9E9 JP (HL)
