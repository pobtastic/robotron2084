CD18: Find Colliding Entity
Used by the routines at Primary_Enemy_Entity_Handler, Clear_Colour_Bar_Column, Process_Tracked_Enemy_Entity, Process_Cluster_Entity, Process_Wall_Bouncing_Enemy and Process_Final_Phase_Enemy_Entity.
Walks the entity list from Entity_List, testing each record's bounds against the current entity via HitTestBounds8AtHL. Returns without carry if no collision is found (list exhausted). Returns with carry set if the current entity's position overlaps another entity's bounds.
