D9EA: Entity State Table
Table of 16-byte entity state records indexed by entity type. Each entry is copied to C521 by Initialise_Entity_State on entity placement. The first two bytes of each record form the dispatch handler address used by Dispatch_Entity_State_Handler; the remaining 14 bytes are parameters for the active effect.
