CB4F: Main Game Loop
Used by the routine at Stage_Next_Wave.
Wave-active game loop entry, called from Stage_Next_Wave once the entity list and playfield are ready. Initialises the player's starting screen position and frame counters, copies the sprite template via Animate_Intro_Screen_Sprites, then falls into the per-frame control poll at Main_Loop_Poll. Each frame: reads the controller via Read_Control_Input, moves and redraws the player sprite, spawns one new entity every three frames adjacent to the player, dispatches all active entity handlers via DispatchWordFromHL, updates the colour bar and protection checks, then loops back to Main_Loop_Poll.
