E4B9: Kill Tracked Enemy Entity
Used by the routine at Process_Tracked_Enemy_Entity.
Kill path reached when no collision-free slot is available for a tracked enemy. Steps back six bytes to the entity position fields, erases the sprite via Erase_Narrow_Entity_Sprite, removes the entity record from the list via Remove_Entity_From_List_Step, and erases the spawn buffer via Erase_Wide_Entity_Sprite. Decrements the active entity count at C42C and the total wave enemy count at Active_Enemy_Count. Initialises a dying-state colour timer via Initialise_Entity_State, writes a replacement Tick_Entity_Colour_Timer entity record at the same position, and scores 10 BCD points. Refreshes the colour bar via Clear_Colour_Bar_Column.
