ECFB: Check S Key During Display
Used by the routines at Display_High_Score_Entry_Screen and Attract_Mode_Loop.
Reads keyboard row FD (A-S-D-F-G) during the high score display loops. Forces the upper three bits high and checks for the S key alone. If S is the only key held, discards the caller's return address and jumps to the tape load continuation handler at PlayerDeath_TapeLoadContinue with the header parameters from C4FD and C4FE. Returns normally if no matching key is detected.
Check_S_Key_During_Display ECFB LD A,$FD Read from the keyboard;
Port Number Bit
0 1 2 3 4
FD A S D F G
ECFD IN A,($FE)
ECFF OR %11100000 Force bits 5-7 high.
ED01 CP $FD Return if the S key is not the only key pressed.
ED03 RET NZ
ED04 POP HL Discard the caller's return address from the stack.
ED05 LD A,($C4FD) Load the tape header byte from C4FD into B.
ED08 LD B,A
ED09 LD A,($C4FE) Load the tape header byte from C4FE into E.
ED0C LD E,A
ED0D JP PlayerDeath_TapeLoadContinue Jump to the tape load continuation at PlayerDeath_TapeLoadContinue.
