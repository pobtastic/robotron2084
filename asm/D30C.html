D30C: Clear Screen and Render HUD
Used by the routine at Stage_Next_Wave.
Clears the screen buffer via ClearScreenBuffer, sets a white background via SetBackgroundColour, then jumps directly to RenderHUD_Draw to render scores and lives for both players. Called by Stage_Next_Wave at the start of each new wave. Unlike RenderHUD this skips redrawing the playfield border and attribute bar.
Clear_Screen_And_Render_HUD D30C CALL ClearScreenBuffer Clear the screen buffer via ClearScreenBuffer.
D30F LD A,$07 Call SetBackgroundColour using WHITE.
D311 CALL SetBackgroundColour
D314 JP RenderHUD_Draw Jump to RenderHUD_Draw to render the scores and lives.
