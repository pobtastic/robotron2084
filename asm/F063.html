F063: Draw Colour Ring
Used by the routine at Draw_Concentric_Rings.
Draws one rectangular colour ring with its top-left corner at column B, row C. The horizontal edges span columns B to FF-B and are drawn via Fill_Row; the vertical edges span rows C to BF-C and are drawn via Fill_Column.
Input
BC Corner offset (B = column, C = row)
Draw the top and bottom horizontal edges.
Draw_Colour_Ring F063 LD H,B Load the corner position into HL and copy the column to A.
F064 LD L,C
F065 LD A,B
F066 SLA A Calculate the horizontal edge length as 0100 minus 2 times the column offset.
F068 NEG
F06A LD D,A
F06B PUSH BC Stash the corner, draw the top edge rightward via Fill_Row, and restore.
F06C CALL Fill_Row
F06F POP BC
F070 LD H,B Set the position to the bottom edge at row BF minus C.
