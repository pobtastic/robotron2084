ECDE: Display Loop Delay
Used by the routines at Display_High_Score_Entry_Screen and Save_Or_Load_High_Scores.
Busy-wait timing delay used by the attract mode and high score display loops. Each call resets B to 00 and spins the inner busy-wait 0100 times; C controls the outer repetition count. Callers set C to 64 before the call, giving 100 outer passes of 256 inner spins each.
Display_Loop_Delay ECDE LD B,$00 Set the inner loop counter to 00 giving 0100 busy-wait spins.
Display_Loop_Delay_Inner ECE0 DJNZ Display_Loop_Delay_Inner Spin until the inner loop counter reaches zero.
ECE2 DEC C Decrement the outer pass count in C and loop back to Display_Loop_Delay_Inner.
ECE3 JR NZ,Display_Loop_Delay_Inner
ECE5 RET Return.
