F5A0: Advance Intro Colour Cycle
Used by the routine at Attract_Mode_Loop.
Advances the intro screen colour cycling effect by one step. Reads source and replacement colours from Intro_Colour_Cycle_Table via the pointer at Intro_Colour_Cycle_Pointer, then scans all 768 attribute cells and replaces every cell matching the source colour with the replacement.
Advance_Intro_Colour_Cycle F5A0 LD HL,($F59E) Load the colour cycle pointer and read the source colour.
F5A3 LD D,(HL)
F5A4 INC HL
Advance_Intro_Colour_Cycle_Read F5A5 LD A,(HL) Read the next cycle entry; jump to Advance_Intro_Colour_Cycle_Apply if not the end.
F5A6 CP $FF
F5A8 JR NZ,Advance_Intro_Colour_Cycle_Apply
F5AA LD HL,$F5B7 Reset the colour cycle pointer to the table start.
F5AD JR Advance_Intro_Colour_Cycle_Read
Advance_Intro_Colour_Cycle_Apply F5AF LD E,A Set up the replacement and source colours.
F5B0 LD A,D
F5B1 LD ($F59E),HL Save the updated colour cycle pointer.
F5B4 JP Advance_Colour_Cycle_Scan Scan the attribute area and apply the colour swap.
