D0F0: Spawn Primary Wave Enemies
Used by the routine at Stage_Next_Wave.
Reads the enemy count and type for the current wave from the definition pointed to by HL, then spawns that many primary wave entities at random valid screen positions by writing records into the entity list at Entity_List_Write_Pointer. Each entity is placed at a random column (1–30) and row (2–22), scaled to pixel coordinates, and drawn to the screen via CC81. If the count is zero the entity list is left empty and the routine returns immediately.
