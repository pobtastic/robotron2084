F5BB: Advance Flash Colour Cycle
Used by the routine at Extra_Life_Flash.
Advances the extra life colour flash display by one step. Scans all 768 attribute cells and replaces each colour with its successor in the cycle table at Flash_Colour_Cycle_Table, rotating the screen through blue, green, cyan, white, yellow, magenta, and red on successive calls.
Advance_Flash_Colour_Cycle F5BB LD DE,$5800 Point to the start of the attribute area.
Advance_Flash_Colour_Cycle_Scan F5BE LD A,(DE) Read the current attribute cell colour.
F5BF LD HL,$F5D2 Point to the colour cycle table.
F5C2 LD BC,$0007 Set the cycle table scan length.
F5C5 CPIR Scan the table for a matching colour entry.
F5C7 JR NZ,Advance_Flash_Colour_Cycle_Next Jump to Advance_Flash_Colour_Cycle_Next if no matching entry was found.
F5C9 LD A,(HL) Replace the attribute cell with the next colour in the cycle.
F5CA LD (DE),A
Advance_Flash_Colour_Cycle_Next F5CB INC DE Advance to the next attribute cell.
F5CC LD A,D Loop back until the entire attribute area has been processed.
F5CD CP $5B
F5CF JR NZ,Advance_Flash_Colour_Cycle_Scan
F5D1 RET Return.
