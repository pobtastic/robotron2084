E262: Find Colliding Wave Enemy
Used by the routines at Handle_Horizontal_Enemy_Entity and Process_Wall_Bouncing_Enemy.
Walks the wave entity list between the snapshot pointer at C50B and the write pointer at C50D. For each entity with a type nibble below 04 tests bounds overlap via HitTestBounds8AtHL; returns without carry set if an overlapping entity is found with its position left in alternate HL. Returns with carry set if the list is exhausted without a match. Entities with a type nibble of 04 or higher are skipped as non-collidable. Called from Handle_Horizontal_Enemy_Entity and Process_Wall_Bouncing_Enemy.
