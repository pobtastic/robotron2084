F6C3: Read Control Input
Used by the routine at Main_Game_Loop.
Reads the active control input for the current control mode stored in *C4FF. Scales the mode index to a word offset, loads the handler address from the table at Control_Handler_Table, pushes Read_Control_Input_Return as the return address, and jumps to the handler. Each handler reads one or more hardware ports and returns the direction and fire bits in A. The common landing pad at Read_Control_Input_Return masks the result to five bits and stores it in B and C.
Output
A Active-low direction and fire bits in the lower five bits
B Copy of A; the F7FE row result in split keyboard mode
C Copy of A; the EFFE row result in split keyboard mode
