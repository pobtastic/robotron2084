C41C 1
Active Enemy Count
Running total of active wave enemies. Incremented as enemies are spawned and decremented as they are killed. When it reaches zero, Stage_Next_Wave loads a fresh wave; when non-zero, the current wave set is re-populated instead. CheckScoreAwardLife_SessionGate also reads it as a session-active gate: non-zero means a game is in progress, so the copy-protection keyboard check is run.

Index of the player's current vertical movement direction. Stored each frame by Main_Game_Loop and used to look up the vertical movement delta and the entity type to spawn above or below the player.

Index of the player's current horizontal movement direction. Stored each frame by Main_Game_Loop and used to look up the horizontal movement delta, the sprite frame, and the entity type to spawn beside the player.

Little-endian pointer loaded from the word table at D3E4 (indexed from D400) when staging a wave after tape load or max-wave reset; copied into Wave_Definition_Pointer when clearing player state in PlayerDeath.

Animation Frame Pair
Initial animation frame pair for the current wave. Written as a word by D1DD and decremented each frame by Main_Game_Loop at CC62 until it reaches zero.
C521 14
Entity State Pointer
Two-byte pointer written by Initialise_Entity_State from the entity state table at Entity_State_Table. Read by Dispatch_Entity_State_Handler and executed via JP (HL) to dispatch the current entity state handler.
C52F 3
Entity State Flag
Tracks the progress of the active entity state effect. 00 = no effect running; 01 = one step complete; 02 = all steps complete. Written by Initialise_Entity_State and polled each frame by Dispatch_Entity_State_Handler and Cycle_Border_From_Screen.
C532 1
Death Sequence Mode
When 01 PlayerDeath skips the intro HUD, "PLAYER" messaging and intro delay before losing a life, and skips the extra "PLAYER" line in the game-over branch. Set from the wave-table path together with the table variant in B.

Countdown loaded to 1A at the start of each wave by Main_Game_Loop. Checked each frame at Main_Loop_Poll; while non-zero the frame is skipped and control branches to the wave-end handler at Clear_Colour_Bar_Column_21.
