Robotron: 2084 disassembly
==========================

A disassembly of the [Spectrum](https://en.wikipedia.org/wiki/ZX_Spectrum) game
[Robotron: 2084](https://en.wikipedia.org/wiki/Robotron:_2084).

Created using [SkoolKit](https://skoolkit.ca).

Browse the latest release:

* [Robotron: 2084 disassembly](https://pobtastic.github.io/robotron2084/) (hexadecimal; mirror [here](http://skoolkit.arcadegeek.co.uk/robotron2084/))
* [Robotron: 2084 disassembly](https://pobtastic.github.io/robotron2084/dec/) (decimal; mirror [here](http://skoolkit.arcadegeek.co.uk/robotron2084/dec/))

To build the current development version, first either install SkoolKit, or
make a clone of the [SkoolKit repository](https://github.com/skoolkid/skoolkit)
and point the environment variable SKOOLKIT_HOME at it. Then:

    $ utils/mkhtml.py
