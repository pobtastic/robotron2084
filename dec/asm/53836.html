53836: Stage Next Wave
Used by the routines at CheckScoreAwardLife and PlayerDeath.
Entry point for wave staging. Resets the entity list write pointer to Entity_List_Base and writes a terminator. If Active_Enemy_Count is non-zero the current wave set is re-populated (Stage_Next_Wave_Populate); otherwise a fresh wave is loaded: enemy parameters are cleared, the next entry in the wave definition table (Wave_Definition_Pointer) is read, the display is reinitialised via Clear_Screen_And_Render_HUD, and the main game loop is entered via Main_Game_Loop with the playfield entity handler 52834 in place.
