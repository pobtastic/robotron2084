60638: Display Loop Delay
Used by the routines at Display_High_Score_Entry_Screen and Save_Or_Load_High_Scores.
Busy-wait timing delay used by the attract mode and high score display loops. Each call resets B to 0 and spins the inner busy-wait 256 times; C controls the outer repetition count. Callers set C to 100 before the call, giving 100 outer passes of 256 inner spins each.
Display_Loop_Delay 60638 LD B,0 Set the inner loop counter to 0 giving 256 busy-wait spins.
Display_Loop_Delay_Inner 60640 DJNZ Display_Loop_Delay_Inner Spin until the inner loop counter reaches zero.
60642 DEC C Decrement the outer pass count in C and loop back to Display_Loop_Delay_Inner.
60643 JR NZ,Display_Loop_Delay_Inner
60645 RET Return.
