36300: Print Dispatch
Used by the routines at PrintMessaging, PrintPlayerScore, PrintWaveNumber, PlayerDeath, Print_BCD_Score, Draw_Score_Entry and Display_High_Score_Screen.
Calls the active print handler stored at PrintState_Handler to process one character or control code, preserving caller context across the call.
PrintDispatch 36300 PUSH BC Stash BC on the stack.
36301 EXX Switch to the shadow registers.
36302 PUSH HL Stash HL and IX on the stack.
36303 PUSH IX
36305 LD IX,32768 Set IX to the print state base at PrintState_Y.
36309 LD HL,36317 Push Print_Dispatch_Return as the handler return address.
36312 PUSH HL
36313 LD HL,(32778) Read the current print handler address from PrintState_Handler.
36316 JP (HL) Jump to the print handler.
