60819: Scan Score Display Attributes
Used by the routine at Display_High_Score_Entry_Screen.
Scans a contiguous block of attribute cells from DE to IX, applying the same colour substitution as Scan_And_Cycle_Screen_Attributes. Called during the name entry phase to keep the score display colours consistent.
Input
DE Start attribute address
IX End attribute address (exclusive)
Scan_Score_Display_Attributes 60819 LD A,(DE) Load the current attribute byte from DE.
60820 LD HL,61019 Set up to search for this attribute in the substitution table at 61019.
60823 LD BC,10
60826 CPIR Search for the attribute in the substitution table.
60828 JR NZ,Scan_Score_Attr_Next Skip replacement if the attribute was not found.
60830 LD A,(HL) Replace the attribute with its substitution byte.
60831 LD (DE),A
Scan_Score_Attr_Next 60832 INC DE Advance to the next attribute cell.
60833 PUSH IX Copy the end address from IX to HL.
60835 POP HL
60836 AND A Clear carry and test whether the range is exhausted.
60837 SBC HL,DE
60839 JR NZ,Scan_Score_Display_Attributes Loop back to Scan_Score_Display_Attributes until all cells in the range are processed.
60841 RET Return.
