50425: Title Strip Blit Parameters
Four bytes read by BlitTitleStrips: *TitleBlit_Params and *50428 bound the strip loop, *50426 selects the screen row / band, *50427 is the run length (LDIR) per strip. Filled by DrawAttractTitle before each BlitTitleStrips call.
TitleBlit_Params 50425 DEFB 0 Strip loop initial index.
50426 DEFB 0 Row / band parameter for the screen row address lookup table.
50427 DEFB 0 Bytes per strip (LDIR count).
50428 DEFB 0 Strip loop terminal index (stop when B reaches this value).
