58000: Spawn Horizontal Wave Enemies
Used by the routine at Stage_Next_Wave.
Entry point called from Stage_Next_Wave. Reads the horizontal enemy spawn count from *HL and stores it at 50219; if zero, returns immediately. Otherwise loops B times, writing a full entity record into the entity list: the 53074 handler address, a random entity type in the range 52-55 a valid random screen position via Pick_Random_Spawn_Position_Y, and trailing bytes 14 13 255. Saves the updated entity list pointer to *Entity_List_Write_Pointer and writes the 53668 list terminator.
