51286: Draw Narrow Entity Sprite
Used by the routines at Main_Game_Loop, Remove_Entity_From_List, Process_Cluster_Entity and Apply_Entity_Velocity.
Entry point for drawing a narrow entity sprite. Sets draw mode (attribute writes enabled) and falls through to Render_Narrow_Entity_Sprite. XOR-renders 8 rows of 2 bytes each; the sprite type is looked up in Sprite_Pixel_Data_Table for the pixel data address, and the entity colour is written to the 2 attribute cells flanking each pixel row.
Input
HL Entity record pointer
