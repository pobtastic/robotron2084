62700: Attract Mode Loop
Used by the routines at GameEntryPoint and Display_High_Score_Entry_Screen.
Clears the playfield, sets the background colour to INK: YELLOW, PAPER: BLACK (BRIGHT) , draws the title screen layout, and loads the strip blit parameters, then counts down 50 frames. Each frame checks for BREAK, S key and tape key presses, advances the title animation and colour cycle, then jumps to the high score entry screen at Display_High_Score_Entry_Screen when the counter reaches zero.
Attract_Mode_Loop 62700 CALL ClearScreenBuffer Clear the playfield and reset sprite state via ClearScreenBuffer.

62719 CALL CheckBreakKey Check for the BREAK key combination via CheckBreakKey.

62746 JP Display_High_Score_Entry_Screen Jump to the high score entry screen at Display_High_Score_Entry_Screen.
