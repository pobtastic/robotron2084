50625: Check Score Award Life
Used by the routine at Main_Game_Loop.
Compares four bytes at Player_1_State with the template at ExtraLifeCompare_b0. Returns if any player byte is below the matching template byte. Otherwise increments *Lives, calls DrawLivesDisplay, adds BCD 2 to *ExtraLifeCompare_b1 with carry into *Lives. If *Active_Enemy_Count is non-zero, tests *65535 and keyboard port 254 then RST 8 (error code 255); if *Active_Enemy_Count is 0 jumps to CheckScoreAwardLife_Continue. At CheckScoreAwardLife_Continue: pop DE, Extra_Life_Flash, Stage_Next_Wave. Used from Erase_Narrow_Entity_Sprite.
