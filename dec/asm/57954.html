57954: Find Colliding Wave Enemy
Used by the routines at Handle_Horizontal_Enemy_Entity and Process_Wall_Bouncing_Enemy.
Walks the wave entity list between the snapshot pointer at 50443 and the write pointer at 50445. For each entity with a type nibble below 4 tests bounds overlap via HitTestBounds8AtHL; returns without carry set if an overlapping entity is found with its position left in alternate HL. Returns with carry set if the list is exhausted without a match. Entities with a type nibble of 4 or higher are skipped as non-collidable. Called from Handle_Horizontal_Enemy_Entity and Process_Wall_Bouncing_Enemy.
