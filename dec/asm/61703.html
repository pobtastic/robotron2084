61703: Print BCD Score
Used by the routines at Draw_Score_Entry and Display_High_Score_Screen.
Prints a BCD-encoded score of B bytes from (HL) as decimal digits on screen, suppressing leading zeros. Each zero nibble advances the column by C without printing; once a non-zero digit is found, all remaining digits are printed. Each digit is positioned using an AT sequence (22) via PrintDispatch; D is the starting column, E is the row, and D advances by C after each digit.
Input
HL Pointer to the BCD score data
B Number of BCD bytes
C Column step between digits
D Starting column position
E Row position
Print_BCD_Score 61703 LD B,4 Set the BCD byte count to 4.
