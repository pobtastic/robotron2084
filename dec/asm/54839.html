54839: Draw Entity Sprite
Used by the routines at Main_Game_Loop, Clear_Colour_Bar_Column, Animate_Intro_Screen_Sprites, Process_Entity_Record_Fast, Begin_Enemy_Death_Animation, Process_Tracked_Enemy_Entity, Kill_Tracked_Enemy_Entity, Process_Cluster_Entity, Process_Player_Tracking_Enemy and Process_Replacement_Enemy_Entity.
Draws or erases an entity's sprite on the display. Draw_Entity_Sprite sets draw mode so the entity colour is written to the attribute area. Erase_Entity_Sprite sets erase mode and skips the attribute write. Both paths enter Draw_Entity_Sprite_Render, which reads the entity type to look up sprite data from Sprite_Pixel_Data_Table, computes the screen position from the Y and X bytes, then XOR-blends twelve pixel rows of sprite data onto the display.
