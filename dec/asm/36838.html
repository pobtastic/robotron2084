36838: Font Configuration Table
Font configuration table, indexed by font number (0–3). Each eight-byte entry comprises six bytes of font metrics copied to 32780–32785 by the select font handler at PrintHandler_SelectFont, followed by a two-byte little-endian pointer to the font bitmap data stored at 32776.
FontConfig_Table 36838 DEFB 4,4,6,2,6,5 Font 0: normal width, narrow width ("I"), wide width ("M"/ "W"), space width, line height, and pixel row count.
36844 DEFB 0,128 Font 0: pointer to font bitmap data.
36846 DEFB 6,5,6,5 Font 1: normal width, narrow width, wide width, and space width.
