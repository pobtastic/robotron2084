58208: Process Tracked Enemy Entity
Per-frame entity handler for IX-indexed tracked wave enemies. If the wave start delay is still counting, advances the entity pointer via Process_Tracked_Enemy_Entity_Advance. Otherwise stores the entity record pointer at Entity_Pointer_Stash, rebuilds the spawn buffer via Build_Entity_Spawn_Record, and searches for a collision- free slot via Find_Colliding_Entity. Tests player contact via HitTestBounds8AtHL and triggers death via Strip_Attributes_And_Fade if touching. Decrements the animation phase, erases the sprite via Erase_Wide_Entity_Sprite, advances the animation frame within the phase mask, then re-enters movement at Process_Tracked_Enemy_Entity_Move. Updates the entity velocity with angular rate via Advance_RNG_Seed and Clamp_Entity_Velocity, clamps position to the play area, rebuilds the spawn buffer, and draws via Draw_Wide_Entity_Sprite. When the animation phase reaches zero, either removes the entity or promotes it to the next phase and spawns a cluster entity with handler Process_Cluster_Entity.
