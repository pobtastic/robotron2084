55748: Initialise Entity State
Used by the routines at Main_Game_Loop, Clear_Colour_Bar_Column, Begin_Enemy_Death_Animation, Kill_Tracked_Enemy_Entity, Kill_Wall_Bouncing_Enemy and Process_Final_Phase_Enemy_Entity.
Activates an entity state effect by copying a 16-byte configuration record from Entity_State_Table into the active state slot at Entity_State_Pointer. The entity type code in E selects the record (multiplied by 16). If Entity_State_Flag is zero, no effect is pending and the routine returns immediately without change.
