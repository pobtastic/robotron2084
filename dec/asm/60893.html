60893: Adjust Name Entry Cursor
Used by the routine at Display_High_Score_Entry_Screen.
Adjusts the cursor within the name entry character field. Searches for the current character position in the table at IX via Search_Score_Entry_Table; if found and within range, updates E, writes INK: WHITE, PAPER: BLACK (BRIGHT) to the new character's attribute cell on the 89 row directly, then updates the previous character's attribute. Simpler than Adjust_Score_Field_Cursor as it addresses the attribute row without calling Calculate_Attr_Cell_Address.
Input

IX Name entry character table
