59343: Spawn Tracked Enemy Entities
Used by the routine at Evaluate_And_Spawn_Tracked_Enemies.
Initialises the tracked-enemy active count at 50221 to zero, copies the per-wave count byte into 50220 and adds it to the total wave enemy count at Active_Enemy_Count, then stashes the wave data pointer. If the count is zero, jumps to Spawn_Tracked_Enemy_Entities_Terminate; otherwise loops B times: writes a 14-byte entity record (handler Process_Tracked_Enemy_Entity, entity type 176 random type variant, random screen position via Pick_Random_Spawn_Position_Y, three initial bytes from the wave data for velocity seeds, and 255 as the terminator) into the entity list via *Entity_List_Write_Pointer, spawning each entity via Draw_Wide_Entity_Sprite. Closes the entity list with the 53668 terminator.
