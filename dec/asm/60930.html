60930: Search Score Entry Table
Used by the routines at Adjust_Score_Field_Cursor and Adjust_Name_Entry_Cursor.
Searches a 3-byte-strided table at HL for the value in A, scanning up to B entries. Returns with the Z flag set and HL pointing to the matched entry if found, or the Z flag clear if not found.
Input
A Byte to search for
B Entry count
HL Table pointer
Search_Score_Entry_Table 60930 CP (HL) Return if the current table entry matches.
60931 RET Z
60932 INC HL Advance HL by 3 bytes to the next table entry.
60933 INC HL
60934 INC HL
60935 DJNZ Search_Score_Entry_Table Decrement the entry count and loop back to Search_Score_Entry_Table.
60937 DEC B Exhaust the search count and return without a match.
60938 RET
