36816: Control Code Dispatch Table
Two-byte handler address table for control codes 16–26 indexed by the control code zero-indexed from 16. Each entry is the address of the parameter handler installed into PrintState_Handler by PrintHandler.
ControlCode_DispatchTable 36816 DEFW PrintHandler_Ink 16: INK colour — parameter handler at PrintHandler_Ink.
36818 DEFW PrintHandler_Paper 17: PAPER colour — parameter handler at PrintHandler_Paper.
36820 DEFW PrintHandler_Flash 18: FLASH — parameter handler at PrintHandler_Flash.
36822 DEFW PrintHandler_Bright 19: BRIGHT — parameter handler at PrintHandler_Bright.
36824 DEFW PrintHandler_Unsupported 20: Unsupported — parameter consumed and discarded via PrintHandler_Unsupported.
36826 DEFW PrintHandler_Unsupported 21: Unsupported — parameter consumed and discarded via PrintHandler_Unsupported.
36828 DEFW PrintHandler_PrintAt_X 22: PRINT AT column — parameter handler at PrintHandler_PrintAt_X.
36830 DEFW PrintHandler_Tab 23: TAB — parameter handler at PrintHandler_Tab.
36832 DEFW PrintHandler_LeftMargin 24: Set left margin — parameter handler at PrintHandler_LeftMargin.
36834 DEFW PrintHandler_RightMargin 25: Set right margin — parameter handler at PrintHandler_RightMargin.
36836 DEFW PrintHandler_SelectFont 26: Select font — parameter handler at PrintHandler_SelectFont.
