60014: Process Replacement Enemy Entity
Per-frame handler for the replacement entity spawned by Handle_Enemy_Collision_Chain. Jumps to Process_Final_Phase_Enemy_Entity if the countdown has already expired. Otherwise locates the associated entity record pointer and jumps to Process_Replacement_Enemy_Entity_Retire to retire if the associated record is in settled state (2). Decrements the countdown and calls Draw_Wide_Entity_Sprite to erase. If the countdown is still non-zero, applies a random X nudge (±7) and spawns via Draw_Wide_Entity_Sprite. If the countdown reaches zero, copies entity data, resets state, and spawns via Process_Replacement_Enemy_Entity_Spawn. When settled, installs the retire handler Process_Entity_Record_Fast.
